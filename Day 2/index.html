Variables

In programming, variables are used to store data values. These values can change during the execution of a program. In most programming languages, variables need to be declared before they can be used.

Example:

var x = 5;
var y = "Hello World!";

Loops

Loops are used to repeatedly execute a block of code until a certain condition is met. There are different types of loops such as for, while, and do...while.

Example:

for (var i = 0; i < 5; i++) {
    console.log("Iteration " + i);
}

Functions

A function is a block of reusable code that performs a specific task. Functions allow you to break down your code into smaller, more manageable pieces.

Example:

function greet(name) {
    console.log("Hello, " + name + "!");
}
greet("Mohit");
